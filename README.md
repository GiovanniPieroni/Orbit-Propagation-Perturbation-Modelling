# Orbit Propagation and Perturbation Modelling

This repository contains a suite of tools for satellite orbit propagation and perturbation modelling, developed as part of the Practical Astrodynamics course.

## Overview

The project is structured around three main components:
1. **Orbit Propagation**: Numerical integration of satellite orbits using Euler and RK4 integrators, validated against SGP4 models.
2. **Perturbation Modelling**: Implementation of various disturbing forces, including:
   - Earth's Oblateness ($J_2$)
   - Solar Radiation Pressure (SRP)
   - Atmospheric Drag
   - Third-Body Perturbations (Sun and Moon)
3. **Initial State Optimization**: Refining the initial state vector to minimize residuals between integrated orbits and SGP4 reference data.

## Project Structure

- `modules/`: Core Python modules containing the physics engines, integrators, and utility functions.
- `run_propagation.py`: Script for basic orbit propagation tasks.
- `run_perturbations.py`: Script for analyzing the effects of different orbital perturbations.
- `run_optimization.py`: Script for performing initial state optimization.
- `data/`: Input and output data files (TLEs, state vectors).
- `plots/`: Visualizations generated by the scripts (Keplerian element evolutions, ground tracks, residuals).

## Features

- **Numerical Integrators**: Custom implementations of Euler and Runge-Kutta 4th Order (RK4) methods.
- **Coordinate Transformations**: Robust transformations between Inertial and Co-rotating frames.
- **Force Modelling**: Modular implementation of gravitational and non-gravitational disturbances.
- **Optimization**: Algorithms (Grid Search, Random Sampling, etc.) to improve orbit fit accuracy.

## Requirements

- Python 3.x
- NumPy
- Matplotlib
- SGP4
- PyYAML

## Usage

Each `run_*.py` script can be executed independently. Ensure that `TLE_GONETSM24.txt` is present in the root directory for satellite data.

```bash
python run_propagation.py
python run_perturbations.py
python run_optimization.py
```
